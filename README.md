# tech-week-2015-crypto

Crypto engine for decoding cipher text.

## Example

```js
var cipher_text;

cipher_text = fs.readFileSync(cipherFile, 'utf8');

console.log(decode(cipher_text));
```


## Installation

```
npm install tech-week-2015-crypto
```
